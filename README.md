# CP2025 — CFGA Grassland Assessment Report 2025
Interactive grassland assessment report for the Canadian Prairie Ecozone.
Prepared by [GrasslandAnalytica](https://www.grasslandanalytica.com) for the Canadian Forage and Grassland Association (CFGA).

## Viewing the Report
Open [index.html](./index.html) or visit the GitHub Pages site.

## Data Products
The raster deliverable (CP2025.tif) and metadata are available in the [Releases](../../releases) section of this repository.

## Map
The interactive map requires an internet connection for the satellite basemap (ESRI World Imagery).
